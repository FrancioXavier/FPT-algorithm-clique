# FPT-algorithm-clique

An FPT approach to the Clique problem parameterized by the maximum degree of the graph.

## Setup and Running

To run the project, ensure you have [Go](https://go.dev/) installed.

### Using Make (Recommended)

- **Run the program:**

  ```bash
  make run
  ```

- **Build the executable:**

  ```bash
  make build
  ```

### Using Go CLI

- **Run directly:**

  ```bash
  go run main.go
  ```

- **Build executable:**

  ```bash
  go build -o fpt-clique main.go
  ```
